### 📋 技术规格

#### 当前MQTT配置
| 配置项 | 新值 | 说明 |
|--------|------|------|
| 数据主题 | `/hi3861/temperature` | 接收温度数据 |
| 命令主题 | `agriculture/cmd` | 发送控制命令（保持不变） |
| 消息格式 | `{"temperature":25.5,"timestamp":1234567890}` | 简化的JSON格式 |

#### 消息处理逻辑
- **温度数据**: 直接使用接收到的值
- **其他数据**: 使用上一次的值或默认值
